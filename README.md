# XorFileProcessor

Программа для модификации файлов бинарной операцией XOR с 8-байтным ключом.

---

## Особенности

- Обработка всех файлов в указанной папке по маске.
- Поддержка перезаписи или сохранения в новую папку с добавлением счётчика.
- Возможность удалять исходные файлы после обработки.
- Режим работы по таймеру с настройкой интервала.
- Асинхронная обработка больших файлов без блокировки интерфейса.
- Отображение прогресса обработки всех файлов через QProgressBar.

---

## Требования

- **Qt 6.10** или выше
- **Компилятор:** MinGW (Windows)
- **Среда разработки:** Qt Creator
- **C++:** Стандарт C++17

---

## Сборка с Qt Creator

1. Открой Qt Creator.
2. Выбери **File → Open File or Project** и укажи файл `XorFileProcessor.pro` или `CMakeLists.txt`.
3. Настрой **Kit** для Qt 6.10 + MinGW.
4. Сборка и запуск проекта:
   - **Build → Build Project**
   - **Build → Run**
